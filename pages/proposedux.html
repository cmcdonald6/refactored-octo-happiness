Please input your time in a 4 digit format (HH:MM). Refresh page to clear.

This is a prototype of my proposed solution using the tools allowed to me by HTML and jQuery. I believe that using a numpad to input your desired time is a far better method compared to the scroll wheel. Apple even started using the numpad on the Alarm tab of the Clock app in a recent update, so it should be a no brainer to make this change to the timer as well.
